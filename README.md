# 🎈 Balloon Game

A fun React + Vite game where players pop balloons, solve math problems, and answer GK questions.

## 🚀 Features
- Balloon popping gameplay
- Math challenges
- GK quiz questions
- Single & battle mode
- Score system + lives
- Particle effects & animations

## 🛠 Tech Stack
- React
- Vite
- JavaScript
- CSS

## 🎮 How to Play
Click balloons to score points.  
Answer questions correctly to earn bonus points.  
Avoid bombs!

## 🌐 Deploy
Built for Vercel / Netlify deployment.
