🎈 Balloon Pop Game

A fun interactive Balloon Pop Game built with React + Vite that mixes fast reflex gameplay with Math and GK (General Knowledge) challenges.
Pop balloons, solve questions, earn points, and survive as long as you can!

🌐 Live Demo: https://balloon-game-mu.vercel.app/

🚀 Features

🎯 Core Gameplay
1.Pop floating balloons to earn points
2.Different balloon types (normal, blue, bomb, diamond, gold)
3.Real-time movement with increasing speed
4.Lives system (game over when lives reach 0)

🧠 Quiz Challenges
1.Random Math Questions ( +, −, ×, ÷ )
2.Fun GK Questions (India + general knowledge)
3.Bonus points for correct answers
4.Game pauses during questions

👥 Battle Mode
1.2-player turn-based mode
2.Player 1 vs Player 2 scoring system
3.Competitive gameplay experience

✨ Extra Effects
1.Particle explosion effects on pop
2.Floating score animations (+2 🎉)
3.Side moving emojis for vibe
4.Shake effect on mistakes
5.Sound effects (pop + background music)

⏸ Game Controls
1.Pause / Resume system
2.Restart anytime
3.Smooth UI interactions

🎮 Game Modes
1.Easy Mode – slower balloons, relaxed gameplay
2.Medium Mode – balanced challenge
3.Hard Mode – fast-paced survival mode
4.Battle Mode – 2-player competitive gameplay

🛠 Tech Stack
   React.js ⚛️
   Vite ⚡
   JavaScript (ES6)
   CSS animations
   HTML5 Audio

# Clone repo
git clone https://github.com/nithyasrimorla-png/balloon-game.git

# Go into project
cd balloon-game

# Install dependencies
npm install

# Start development server
npm run dev

🚀 Deployment
This project is deployed using Vercel.

##To deploy your own version:

1.Push code to GitHub
2.Import repo in Vercel
3.Select React + Vite
4.Click Deploy
5.Done ✅
