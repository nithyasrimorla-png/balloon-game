# 🎈 Balloon Pop Game

A fun interactive Balloon Pop Game built with React + Vite that mixes fast reflex gameplay with Math and GK (General Knowledge) challenges. Pop balloons, solve questions, earn points, and survive as long as you can!

🌐 **Live Demo:** (https://balloon-game-mu.vercel.app/)

## 🚀 Features

### 🎯 Core Gameplay
- Pop floating balloons to earn points
- Different balloon types (normal, blue, bomb, diamond, gold)
- Real-time movement with increasing speed
- Lives system (game over when lives reach 0)
- Limited time

### 🧠 Quiz Challenges
- Random Math Questions (+, −, ×, ÷)
- Fun GK Questions (India + general knowledge)
- Bonus points for correct answers
- Game pauses during questions

### 👥 Battle Mode
- 2-player turn-based mode
- Player 1 vs Player 2 scoring system
- Competitive gameplay experience

### ✨ Extra Effects
- Particle explosion effects on pop
- Floating score animations (+2 🎉)
- Side moving emojis for vibe
- Shake effect on mistakes
- Sound effects (pop + background music)
- sound effects for every correct &wrong answer

### ⏸ Game Controls
- Pause / Resume system
- Restart anytime
- Smooth UI interactions

### 🎮 Game Modes
- **Easy Mode** – slower balloons, relaxed gameplay
- **Medium Mode** – balanced challenge
- **Hard Mode** – fast-paced survival mode
- **Battle Mode** – 2-player competitive gameplay

### Choose character
  - Cat,dog,fox,panda
  - When a balloon is popped, the selected character (cat) is used to generate customized particle effects. These include character-themed particles, creating a       more engaging and visually dynamic experience.


## 🛠 Tech Stack
- React.js ⚛️
- Vite ⚡
- JavaScript (ES6)
- CSS animations
- HTML5 Audio

## Getting Started

```bash
# Clone repo
git clone https://github.com/nithyasrimorla-png/balloon-game.git

# Go into project
cd balloon-game

# Install dependencies
npm install

# Start development server
npm run dev

## 🚀 Deployment

This project is deployed using [Vercel](https://vercel.com).

To deploy your own version:
1. Push code to GitHub
2. Import repo in Vercel
3. Select React + Vite
4. Click Deploy
5. Done
```
